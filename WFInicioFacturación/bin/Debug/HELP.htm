Inicio de Sesión

En este módulo puede acceder el administrador con su Usuario y Contraseña, los cuales por defecto el Usuario será admin y la contraseña será adminPALE

 

 

 

Recuperación de Contraseña

En este módulo puede el administrador recuperar su contraseña mediante correo solo debe ingresar correctamente la respuesta a la pregunta asignada y luego ingresar el correo a donde se desea enviar la contraseña.

 

 

 

 

 

Página Principal del Administrador

En este módulo se puede interactuar de 6 maneras diferentes las cuales son:

1.    Barra Superior Izquierda: En este apartado podrá ocultar o mostrar las distintas opciones a elegir

2.    Cliente: En este apartado podrá ingresar al crud (sección de ingresos, eliminaciones, modificaciones y buscar) del cliente

3.    Alimentos: En este apartado podrá ingresar al crud (sección de ingresos, eliminaciones, modificaciones y buscar) de los alimentos

4.    Reportes: En este apartado podrá mostrarnos los reportes de clientes y alimentos.

5.    Nueva orden: En este apartado nos enviara a una sección donde podremos ingresar los productos que el cliente desee

6.    Actualizar: En este apartado se          actualizará de forma inmediata la tabla donde se nos visualizará las ordenes recientes

 

 

 

 

 

 

 

 

 

 

 

Página Principal del Administrador: Inicio - Cliente

En este módulo se puede interactuar de 4 maneras diferentes las cuales son:

1.    Agregar: Aquí el administrador podrá agregar a un nuevo cliente si este no está en el sistema

2.    Buscar: En este apartado el administrador podrá buscar al cliente para verificar si está en el sistema

3.    Modificar: En este apartado el administrador podrá corregir de manera inmediata algún dato mal ingresado

4.    Eliminar: En este apartado el administrador podrá eliminar cualquier cliente que desee teniendo en cuenta que todos sus datos se eliminaran

5.    Salir: En este apartado cerrara la parte del crud de cliente para pasar a la página principal del administrador

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Página Principal del Administrador: Agregar - Cliente

En este módulo el administrador agregará un nuevo cliente agregando en cada apartado los datos necesarios del cliente y para finalizar este deberá dar clic en añadir y estará inmediatamente en el sistema

 

 

Página Principal del Administrador: Buscar - Cliente

En este módulo el administrador buscará un cliente para verificar si está en el sistema o no, primero deberá ingresar la cedula del cliente y luego en buscar donde se le mostrara una pantalla con los datos dando en aceptar para cerrar o sino un mensaje de No se ha encontrado

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 


Página Principal del Administrador: Modificar - Cliente

En este módulo el administrador tratara de modificar los datos de un cliente, donde se deberá buscar el cliente por su cedula y al encontrar al cliente se mostrará sus datos donde se deberá corregir el campo incorrecto y al finalizar la edición se deberá de confirmar para que se guarden los datos y salir a la página de inicio del cliente

 

 

 

Página Principal del Administrador: Eliminar - Cliente

En este módulo el administrador tratara de eliminar los datos de un cliente, donde se deberá buscar el cliente por su cedula y al encontrar al cliente se mostrará sus datos donde deberá de confirmar para que se eliminen los datos y salir a la página de inicio del cliente

 

 

 

 

Página Principal del Administrador: Inicio - Producto

En este módulo se puede interactuar de 4 maneras diferentes las cuales son:

6.    Agregar: Aquí el administrador podrá agregar a un nuevo producto si este no está en el sistema

7.    Buscar: En este apartado el administrador podrá buscar al producto para verificar si está en el sistema

8.    Modificar: En este apartado el administrador podrá corregir de manera inmediata algún dato mal ingresado

9.    Eliminar: En este apartado el administrador podrá eliminar cualquier producto que desee teniendo en cuenta que todos sus datos se eliminaran

10.  Salir: En este apartado cerrara la parte del crud de productos para pasar a la página principal del administrador

 

 

Página Principal del Administrador: Agregar - Producto

En este módulo el administrador agregará un nuevo producto agregando en cada apartado los datos necesarios del producto y para finalizar este deberá dar clic en añadir y estará inmediatamente en el sistema, también tenemos una sección donde se podrá buscar el producto para poder añadirlo

 

 

Página Principal del Administrador: Buscar - Producto

En este módulo el administrador buscará un cliente para verificar si está en el sistema o no, primero deberá ingresar el id del producto y luego en buscar donde se le mostrara una pantalla con los datos y una imagen, dando en aceptar para cerrar o sino un mensaje de No se ha encontrado

 

 

 

 

Página Principal del Administrador: Modificar - Producto

En este módulo el administrador tratara de modificar los datos de un producto, donde se deberá buscar por su id y al encontrar al producto se mostrará sus datos donde se deberá corregir el campo incorrecto y al finalizar la edición se deberá de confirmar para que se guarden los datos y salir a la página de inicio del producto

 

 

Página Principal del Administrador: Eliminar - Producto

En este módulo el administrador tratara de eliminar los datos de un producto, donde se deberá buscar por su id y al encontrar al producto se mostrará sus datos donde deberá de confirmar para que se eliminen los datos y salir a la página de inicio del producto

 

 

 

 

Página Principal del Administrador:  Selección de Productos

En este módulo el administrador elegirá de un catálogo de productos los que el cliente desee ordenar, encontrados 3 secciones para interactuar las cuales son:

1.    Cantidad: En este apartado se nos dará a elegir una cierta cantidad en números de productos a ordenar

2.    Añadir: En este apartado se añadirán los productos que vayamos eligiendo y se nos ira mostrando en el apartado de abajo para verificar si se añadió o no.

3.    Facturación Icono Inferior Derecho: En este apartado se nos dirigirá a una facturación donde se deberán ingresar los datos del cliente

 

 

 

Página Principal del Administrador: Facturación

En este módulo el administrador deberá ingresar los datos del cliente para facturar como desee ya sea como C/F o en su nombre, pidiendo su cedula y la forma de pago ya sea en efectivo o tarjeta en donde se ingresara el dinero que se le da al administrador y mostrándonos cuanto se debe pagar pero si es en tarjeta se deberá ingresar la clave de la tarjeta con su número de tarjeta y de este modo se pagara dependiendo el modo dando en terminar, recordando que en apartado de abajo se detallara los productos que se van a pagar.
